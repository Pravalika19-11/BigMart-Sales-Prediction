# 📊 BigMart Sales Prediction

This project predicts product sales for BigMart stores using machine learning techniques and provides a web interface for real-time predictions.

---

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Multiple regression models:
  - Linear Regression
  - Ridge Regression
  - XGBoost
- Model evaluation and comparison
- Flask-based web application for predictions

---

## 🛠 Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- XGBoost  
- Flask  
- HTML, CSS  

---

## 📊 Project Workflow
1. Data Cleaning & Preprocessing  
2. Exploratory Data Analysis  
3. Feature Engineering  
4. Model Building  
5. Model Evaluation  
6. Deployment using Flask
